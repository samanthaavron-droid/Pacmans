# Design Doc

*Core Loop*: Surviving a 3D labyrinth by avoiding enemies and collecting triangles
*Goal*: Getting all of the triangles before escaping through a portal
*Challenge*: Enemies move along the labyrinth searching for the player
*Feedback*: Player knows how many triangles there is left to collect
*Progress*: More geometric levels

*Theme*: Dimensions
*Setting*: Floating world
*What do you collect*: Triangles
*New Mechanic*: The game is secretly 3D
*How does it work*: The player's camera is aligned with the current plane and changes depending on their plane
